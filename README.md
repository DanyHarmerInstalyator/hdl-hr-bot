# HDL HR Bot — Telegram бот для автоматизации HR-процессов

### О проекте
Корпоративный Telegram-бот, созданный для отдела персонала (HR) компании HDL. Автоматизирует процессы онбординга новых сотрудников, сбора заявок и предоставления справочной информации.

Бот решает проблему "бумажной рутины" и потери информации в мессенджерах, создавая единое окно коммуникации между HR и сотрудниками.

---

### Функционал

**Для сотрудников:**
- 👤 **Онбординг новых сотрудников** — автоматическая выдача приветственных материалов, инструкций и доступов
- ❓ **FAQ** — база ответов на частые вопросы (документы, регламенты, контакты)
- 🔍 **Умный поиск** — поиск по базе знаний с учетом контекста и синонимов
- 📄 **Загрузка документов** — доступ к шаблонам заявлений, бланкам и инструкциям

**Для HR-отдела:**
- 📊 **Сбор аналитики** — учет запросов, выявление частых проблем
- 🗂️ **Структурированное хранение** — все заявки и обращения сохраняются в базе данных
- 🔔 **Уведомления** — оповещения о новых заявках от сотрудников

---

### Стек технологий

- **Язык:** Python 3.10+
- **Фреймворк:** Aiogram 3.x
- **База данных:** SQLite / PostgreSQL (через database.py)
- **Деплой:** Render.com (поддержка webhook и polling)
- **Дополнительно:** 
  - Асинхронная архитектура
  - Обработка контекста диалога
  - Интеграция с внешними API (при необходимости)

---

### Бизнес-ценность

✅ **Сокращение времени HR-специалистов** — автоматизация ответов на типовые вопросы  
✅ **Прозрачность процессов** — все заявки в единой базе, ничего не теряется  
✅ **Быстрый онбординг** — новые сотрудники получают всю информацию сразу, без ожидания  
✅ **Аналитика** — понимание, какие вопросы возникают чаще всего, для улучшения процессов  

---
### Запуск проекта

1. Клонировать репозиторий
2. Установить зависимости: `pip install -r requirements.txt`
3. Создать файл `.env` с токеном бота
4. Запустить: `python main.py`
